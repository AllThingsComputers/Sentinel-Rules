// Title of Detection:   Mimikatz  dump in command-line
// Authorship & Contact: https://allthingscomputers.medium.com  https://twitter.com/allthingshandle  https://allthingscomputersblog.wordpress.com/ 
 // Description of the Detection: From article of Common Mimikatz dumping commands  
// References (if available): 
// The Logic of the Detection or Query Begins Below This Line
SysmonEvent
| where EventID == 1
| where CommandLine matches regex  '(?i)(Isadump::sam|Isadump::Isa /patch)'
