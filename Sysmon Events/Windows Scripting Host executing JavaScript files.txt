// Title of Detection:  Windows Scripting Host executing JavaScript files
// Authorship & Contact: https://allthingscomputers.medium.com  https://twitter.com/allthingshandle  https://allthingscomputersblog.wordpress.com/ 
 // Description of the Detection:  This detection analytic will identify unusual activity originating from wscript.exe executing JavaScript files from the %APPDATA% directory. This applies to Gootkit because the initial loader for the threat is implemented in JavaScript that gets executed via wscript.exe when the victim double-clicks on the downloaded loader 
// References (if available):  p57  https://resource.redcanary.com/rs/003-YRU-314/images/2022-ThreatDetectionReport-RedCanary.pdf
// The Logic of the Detection or Query Begins Below This Line
SysmonEvent
| where EventID == 1 and Image endswith "wscript.exe" and matches regex "(?i)\\APPDATA\\'
