// Title of Detection: HttpOpenRequest Potential Download
// Authorship & Contact: https://allthingscomputers.medium.com  https://twitter.com/allthingshandle  https://allthingscomputersblog.wordpress.com/ 
 // Description of the Detection:  HttpOpenRequest function can download adversary tools
// References (if available):    p65      https://resource.redcanary.com/rs/003-YRU-314/images/2021-Threat-Detection-Report.pdf 
// The Logic of the Detection or Query Begins Below This Line
SysmonEvent
| where EventID == 1
| where CommandLine matches regex 'HttpOpenRequest'
